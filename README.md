# Modern Ecommerce Platform

A premium, high-performance ecommerce web application built with a focus on modern aesthetics, responsiveness, and SEO optimization.

## 🚀 Features

- **Modern UI/UX**: Built with React 19 and Tailwind CSS 4 for a sleek, premium look.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewports.
- **SEO Optimized**: Dynamic meta tags, structured data, and semantic HTML for better search engine ranking.
- **Product Management**: Comprehensive product pages, collections, and filtering systems.
- **Shopping Experience**: Fully functional shopping cart, order placement, and order history.
- **Authentication**: Secure login and sign-up flows.
- **Performance**: Optimized for speed and accessibility using Vite.

## 🛠️ Tech Stack

- **Frontend**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Routing**: [React Router 7](https://reactrouter.com/)
- **Build Tool**: [Vite 8](https://vitejs.dev/)
- **Notifications**: [React Toastify](https://fkhadra.github.io/react-toastify/)
- **Icons**: Lucide React (or equivalent)

## 🏁 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (Latest LTS recommended)
- [npm](https://www.npmjs.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```bash
   cd ecommerce-with-mern
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally

To start the development server:
```bash
npm run dev
```
The application will be available at `http://localhost:5173`.

## 📦 Scripts

- `npm run dev`: Starts the Vite development server.
- `npm run build`: Builds the application for production.
- `npm run preview`: Locally previews the production build.
- `npm run lint`: Runs ESLint to check for code quality issues.
